# Movie Images Setup Instructions

## Directory
Save all images to: `frontend/public/images/`

## Movie Images (5 files needed)

1. **Inception** 
   - Filename: `inception.jpg`
   - From: Inception movie poster (first image provided)

2. **The Dark Knight**
   - Filename: `the-dark-knight.jpg`
   - From: The Dark Knight movie poster (second image provided)

3. **Interstellar**
   - Filename: `interstellar.jpg`
   - From: Interstellar movie poster (third image provided)

4. **The Room**
   - Filename: `the-room.jpg`
   - From: The Room movie poster (fourth image provided)

5. **Titanic**
   - Filename: `titanic.jpg`
   - From: Titanic movie poster (fifth image provided)

## How to Use

Once the images are saved in `frontend/public/images/`:

1. Your app will automatically display these images when you add movies with these exact titles
2. The images must match the titles exactly as shown above
3. Movies with other titles will use placeholder images

## Quick Test

1. Open http://localhost:3000 in your browser
2. Click "+ Add Movie"
3. Add a movie titled "Inception" and set other fields
4. The Inception poster should display in the movie card
